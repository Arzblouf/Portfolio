Projet 1 : Chiffrement de données

▼

Description du projet

Suite à une demande client, une application du client devait être mise à jour afin de répondre aux nouvelles réglementations du RGPD en matière de protection des données à caractère personnel.
Mon rôle était de réaliser une ébauche de code permettant de chiffrer les informations avant qu'elles ne soient stockées dans la base de données, ainsi que de les déchiffrer lors de l'affichage dans l'application.

L'entreprise

Normasys - Entreprise de services informatiques
Normasys est une entreprise fondée en 1998 et spécialisée dans les services informatiques aux organisations, plus particulièrement les organisation bancaires et d'assurance comme Groupama.

Technologies utilisées

  • Visual Studio Community 2022
  • C#
  • Bibliothèque MySQL C#
  • Microsoft Azure

Projet 2 : Mise à jour de version

▼

Description du projet

Dans le cadre d'un scan de sécurité effectué par le client, il a été demandé à l'entreprise de mettre le site web à jour afin de répondre aux attentes du client. Dans ce but, l'équipe de développeurs associée à ce site a décidée de mettre à jour la version de JQuery qu'ils utilisaient afin de réduire les risques de sécurité en passant à une version plus récente qui profite d'une meilleurs sécurité.
Mon rôle était d'aider les développeurs à vérifier le code de l'application pour y modifier toutes les fonctions qui pourraient être obsolète ou incompatible avec la nouvelle version de la bibliothèque et ainsi minimiser les risques de sécurité.

L'entreprise

Normasys est une entreprise fondée en 1998 et spécialisée dans les services informatiques aux organisations, plus particulièrement les organisation bancaires et d'assurance comme Groupama.
