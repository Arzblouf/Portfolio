Projet 1 : Galactic Cube

▼

Description du projet

Il s'agit d'un mini-jeu de type "shoot'em up" fait en Python sur l'IDE Pycharm. Le projet utilise le module Python Pyxel permettant des rendus 2D simples et rapides. Dans ce mini-jeu, le but est de vaincre le maximum d'ennemis possibles pour obtenir le plus gros score sans perdre toutes ses vies.

Technologies utilisées

  • Python
  • Pyxel
  • Pycharm
